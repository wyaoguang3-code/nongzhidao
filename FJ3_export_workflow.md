# FJ3 外銷數據更新操作檔（TradeCoa）

> 目的：更新 `code_FJ3_export.json`，供頁面外銷圖表（近5年月趨勢、近20年年趨勢）使用。

## 一、查詢頁面
- 網址：`https://agrstat.moa.gov.tw/sdweb/public/trade/TradeCoa.aspx`
- 功能：農產品別(COA)資料查詢

## 二、固定查詢條件
1. 資料期間：`105/01` ～ 最新月份（目前可到 `115/03`）
2. 資料週期：`年月`
3. 進出口別：`出口`（僅勾出口）
4. 資料屬性：勾 `量`、`值`；幣別選 `新台幣`
5. 國別代號：`G0,G1,G2,G3,G4,G5,G6,G7,G8,G9`
6. 農產品別代號：`103##13`

## 三、排列類型（重要）
- 行：`進出口別`
- 列：`資料期間`、`農產品別`、`國別`

> 注意：列設定必須用箭頭套用：
> 先點右側清單中的項目（如農產品別/國別）→ 再按同一列的左箭頭（←）移入左側列清單。

## 四、產出流程
1. 按「呈現報表」
2. 按「下載 Ods檔」
3. 將 ODS 解析為結構化資料

## 五、JSON 目標格式
檔案：`publish/data/code_FJ3_export.json`

必要欄位：
- `latest_month`
- `monthly5`（近5年，每月）
- `yearly`（2016～最新，每年）
- `avg_price_ntd_per_ton = value_ntd / weight_ton`

## 六、圖表規格
1. 每月趨勢圖：近 5 年（60 筆）
2. 每年趨勢圖：2016～最新（年彙總）
3. 單位：
   - 量：`ton`
   - 值：`NTD`
   - 均價：`NTD/ton`

## 七、失敗保護
- 若來源站回應 500/連線失敗：
  - 不覆蓋既有 `code_FJ3_export.json`
  - 記錄狀態到 `data/fj3_update_status.json`

- 以 GitHub Actions 定時執行
- 更新時段：08:00～13:00 每15分鐘
- 流程：抓取 → 解析 → 驗證非空 → 寫檔 → commit/push
